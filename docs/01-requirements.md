# 需求与约束（一期）

## 产品目标

做一个可快速上线的 H5「谁是卧底发牌器」，用于线下聚会开局与投票后的身份查验。

## 关键玩法约束

- **玩家查看阶段**
  - 玩家只看见自己的“词”，不应知道自己身份（卧底/平民）。
  - 交接必须防偷看：默认隐藏，长按后揭晓，松开立即隐藏。
- **发牌完成后**
  - 手机主要在房主手上。
  - 房主根据投票结果对某位玩家进行“身份查验”：只显示“卧底/平民”，**不显示词**（房主也是玩家）。
  - 支持低频场景“忘词回看”：把手机交给该玩家，长按只显示该玩家的词。

## 一期范围（MVP）

- 开局配置：玩家人数（4–20），卧底人数自动计算
- 词对：内置词库 + 自定义词对（本机 localStorage）
- 发牌：按序逐个玩家查看（翻牌 + 长按揭晓）
- 房主：玩家面板 + 快捷长按查验（进度完成揭晓并标记）+ 忘词回看（弹层）
- 胜负提示：根据“已查验=淘汰”规则提示游戏继续/结束与胜利方

## 暂不做 / 后续改进

- 白板、多人阵营等扩展规则
- 查验“撤销/改位/清空记录”等管理功能
- 多人共享/联机、房间号、远端词库
