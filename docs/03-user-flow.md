# 用户流程（玩家视角 / 房主视角）

## 开局（房主）

1. 打开 `/setup`
2. 调整玩家人数（中央大数字 + +/-）
3. 选择词对（推荐随机）
4. 点击“开始发牌”

## 发牌（玩家逐个查看）

1. 进入 `/reveal/1`
2. 手机交给第 1 位玩家
3. 玩家长按揭晓自己的词，松开隐藏
4. 玩家点击“交给下一位”，进入 `/reveal/2`
5. 重复直到最后一位玩家
6. 自动进入 `/host`

## 游戏中（描述与投票，房主持机）

### 快捷查验（高频）

1. 在 `/host` 玩家面板，长按某个玩家格子
2. 出现整块进度画布，松手取消
3. 进度完成后显示“卧底/平民”，并在格子上永久标记

### 弹层操作（低频/精确操作）

- 单击玩家格子打开弹层
  - “身份查验”：长按翻牌揭晓身份（仅卧底/平民），完成后标记到格子
  - “忘词回看”：把手机交给该玩家，长按翻牌仅展示词，松开隐藏

## 胜负提示（一期假设）

- 一期默认假设：**被查验的玩家视为“被票出并淘汰”**。
- 在此假设下：
  - 存活卧底数为 0 → 平民胜利
  - 存活卧底数 ≥ 存活平民数 → 卧底胜利

如果你的线下规则不同，需要调整胜负判定逻辑。
